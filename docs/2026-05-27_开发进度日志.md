### 1. AI 情感解读缓存问题修复
- **问题**：Python 端调用 Server API（/api/ai-insight）返回中文内容，即使 lang=en
- **排查过程**：
  - 用 curl 直接测试 Server API → 成功返回（但 cached: true）
  - 检查 Python 端代码 → lang 参数传递正确
  - 发现 Server 没有被成功 kill 再启动（旧进程残留）
  - kill 旧进程后重启 Server → cached: false，法语内容正确返回
- **涉及文件**：`server/index.js`、`kindredsouls-voice-v2.py`
- **修复方法**：确保 kill 后重启 Server，清空内存缓存

## 二、产品规划讨论

### 1. 社交模块讨论
- 山子大叔提出两个新功能想法
- 最终方案确定（详见 `docs/KindredSouls_社交模块规划.md`）：
  - **瞬时分享（阅后即焚）**：3 分钟自动销毁，端到端加密，Phase 3 排期
  - **语音祝福卡片**：选声线+写文案生成语音卡片（不做声音克隆），Phase 2.5 排期

### 2. 声音克隆方案决策
- 山子大叔建议：AI 模拟对方声音给对方分享祝福
- 最终决策：**不做真声音克隆**，改为"选声线 + 写心意"轻量方案
- 原因：伦理风险、consent 机制、成本（ElevenLabs $5-25/次）
- 真声音克隆留给未来付费功能（Phase 4+）

### 3. 免费/付费策略
- 瞬时分享 & 语音祝福卡片：上线后免费 3-6 个月
- 收费触发：DAU ≥ 10,000 或免费期满
- 收费模式：月订阅 $4.99-9.99 + 按次付费 $0.99/次 + 情侣双人包 $7.99/月
- 转化率目标：≥ 8%
- 老用户前 1000 名永久 5 折
